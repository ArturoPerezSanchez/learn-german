# Feedback States

Transparent PNG stickers for answer feedback, rewards, hints, and short celebration moments.

## Files

```text
feedback-states-sheet.png
feedback-states-sheet-chromakey.png
feedback-states-manifest.json
feedback-states-contact-sheet.jpg
items/
  correct-burst.png
  wrong-gentle-hint.png
  try-again-sparkle.png
  streak-star-flame.png
  level-complete-trophy.png
  perfect-score-badge.png
  hint-glow-lightbulb.png
  review-complete-stack.png
```

Use the files in `items/` directly in the app. Keep `feedback-states-sheet-chromakey.png` only as source material.

## Suggested Uses

- `correct-burst.png` - correct answer moment.
- `wrong-gentle-hint.png` - gentle incorrect-answer feedback.
- `try-again-sparkle.png` - retry prompt or second attempt.
- `streak-star-flame.png` - streak or repeated-correct feedback.
- `level-complete-trophy.png` - game or lesson complete screen.
- `perfect-score-badge.png` - no-mistake result.
- `hint-glow-lightbulb.png` - hint reveal or grammar clue.
- `review-complete-stack.png` - review queue complete or flashcard session done.

All assets are transparent PNGs and intentionally have no readable text, so UI copy can be localized separately.
