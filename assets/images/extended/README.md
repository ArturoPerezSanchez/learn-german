# Extended App Assets

This generated pack contains 532 lightweight SVG assets for common German learning app UI needs.

## Preview

Open `index.html` to browse the pack.

## Categories

- `achievements` - 100 assets
- `lessonBadges` - 96 assets
- `grammarArticleBadges` - 24 assets
- `grammarCaseBadges` - 24 assets
- `pronounChips` - 36 assets
- `sentenceTiles` - 136 assets
- `feedbackParticles` - 80 assets
- `uiAccents` - 36 assets

## Regeneration

Run:

```powershell
python assets/scripts/generate_extended_app_assets.py
```

The generator uses project design tokens and writes deterministic SVG assets under `assets/images/extended/`.
