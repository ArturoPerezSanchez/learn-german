# Main Menu Assets

These assets are meant to make the current game selection menu feel more polished without forcing a new layout.

## Recommended Uses

- `main-menu-background-town.png` - full-screen background behind the existing card grid.
- `hero-decoration.png` - transparent overlay for the top hero panel, with the center left open for the title.
- `icons/*.png` - replacement game card icons.
- `stickers/*.png` - small accents for empty areas, progress moments, section headers, or hover states.
- `section-ribbon-vocabulary.svg` and `section-ribbon-grammar.svg` - optional section heading accents.
- `card-corner-vocabulary.svg` and `card-corner-grammar.svg` - optional subtle card corner overlays.
- `footer-progress-band.svg` - optional bottom decoration behind the vocabulary count.

## Manifests

```text
main-menu-assets-manifest.json
game-mode-icons-manifest.json
decor-stickers-manifest.json
```

## Review

Open `main-menu-assets-contact-sheet.jpg` to review the whole pack quickly.

The PNG assets were generated with a chroma-key workflow and saved as transparent PNGs where appropriate. The `*-chromakey.png` files are source material and do not need to be loaded by the app.
